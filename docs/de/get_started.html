    • Erste Schritte¶

    1. QGIS 3 herunterladen und installieren.

    2. Das Plugin „QGIS Model Baker“ mit über die Erweiterungsverwaltung installieren.

    3. Das Menü Datenbank –> QGIS Model Baker –> Erzeugen anklicken.

    4. Die Datenbank und - falls zutreffend - Interlis konfigurieren
