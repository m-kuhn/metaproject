    • QGIS Projektgenerator¶

    QGIS Projektgenerator ist ein Plugin für QGIS 3.

    Es reduziert den Initialaufwand für die Konfiguration von QGIS Projektdateien. Dafür werden Informationen von PostgreSQL Datenbanken oder Interlis Modellen verwendet.
