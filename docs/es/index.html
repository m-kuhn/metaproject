    • QGIS Model Baker for QGIS¶

    QGIS QGIS Model Baker is a QGIS plugin for QGIS 3.

    Reduce la configuración inicial para archivos de proyecto de QGIS por medio de una lectura de información de base de datos en PostgreSQL o archivos de Interlis.
