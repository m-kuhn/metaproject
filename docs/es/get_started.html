    • Para iniciar¶

    1. Instale QGIS 3, hasta que haya una versión oficial de QGIS 3, instale QGIS 2.99 desde OSGeo4W (windows) o una fuente de acuerdo al sistema operativo que utilice.

    2. Use el administrador de plugins para instalar el plugin QgsProjectGenerator

    3. Dé click en el menú Base de datos –> Project Generator –> Generar

    4. Configure su base de datos y, si aplica, la configuración de Interlis
