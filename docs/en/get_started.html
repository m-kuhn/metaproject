    • Get Started¶

    1. Install QGIS 3.

    2. Use the plugin manager to install the “QGIS Model Baker” plugin

    3. Click the Database –> QGIS Model Baker –> Generate menu

    4. Configure your database and, if applicable, Interlis configuration
